# 京东-东东农场

## 功能说明

- 实现简单的京东东东农场简单自动任务功能。
- ⚠️ 注意:使用此快捷指令需要账号的 Cookie，指令为本地运行任务，不会上传 Cookie 在内的任何隐私信息，请确保下载途径的安全性。
- 后台模式请用桌面小组件方式运行
- 兼容 ios13-15、ipad、多账号、自动化
